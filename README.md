# Risk Adaptive Passwordless Authentication System

## Overview
This project is a secure authentication system that uses machine learning and risk analysis to provide passwordless login.

## Features
- Passwordless Authentication
- Risk-Based Access Control
- Machine Learning Analysis
- OTP Verification
- User Activity Monitoring

## Technologies Used
- Python
- Flask
- HTML/CSS/JavaScript
- Machine Learning
